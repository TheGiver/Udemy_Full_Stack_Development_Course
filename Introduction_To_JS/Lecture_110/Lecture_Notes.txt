Lecture Title: JS Variables
---------------------------
The prompt():
This is another JS function that behaves in a similar way to the 
alert(), except that it allows for user input. 

--> Usage:
prompt("Message");

--> Storing user input:
If you would like to store the user input, then you would have to make 
use of what is known as variables. 
---------------------------
Defining a variable:
A variable can be defined in the following way:
var varName = data

examples:
var variableName = "Hello";
var variableName = 3;
